# communicable-LAN
communicable-LAN (Or c-LAN) is an infectious Metasploit payload I started working on on ```7/7/2026```


## What is its purpose? 
c-LAN's purpose is to serve as a payload that allows private network devices to be infected by an already compromised machine on the same network, enabling those devices to be used as a botnet for DDoS attacks against systems with weak monitoring, such as many IPs attacking without provider checks.
